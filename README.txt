Код автоматизирует процесс обработки данных в Excel:
  1.	Выбирает файл-исходник «Заявки ЕИССД»
  2.	Убирает верхние строки до таблицы 
  3.	По колонке «Тип операции» - фильтр «Смена адреса»
  4.	По колонке «Причина отказа» - фильтр «Нет связи с клиентом»
  5.	Оставляет колонки: Дата/время приема, Адрес, Клиент, Телефон, - остальные удаляет
  6.	Колонка «Телефон» убирает слова «не верифицирован»
  7.  Сохранение в файл .xlsx
